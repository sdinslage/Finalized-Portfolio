Colorblind Site

This is another project that I worked on in a group. For this project, I was responsible for creating a colorblind test that users could take to see if they had any type of color blindness. I created the entirety of the test itself, with the homepage being created by my two groupmates. I created the images for the test via this website. I wrote scripts for the test that as well. The main script counts the amount of errors the user makes. If the user gets 3 questions wrong that are labeled as red-green, the results page will say they might have red-green color blindness, and vice versa with yellow-blue. If there are a total of 6 or more errors, the test results will say the user may have monochromia, or can only see black and white. There are also 2 control questions that are in black and white. If the user gets either one of those wrong, the test will declare the user may have been randomly guessing as black and white should be visible to all. This was a fun project to work on, I myself am not color blind but I know people who are and it was fun creating a test and having them try it to see if my test was accurate.
